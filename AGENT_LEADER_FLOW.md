## 角色定位

| Agent | 角色 | 核心任務 |
|-------|------|----------|
| **Agent Leader** | 首席策略官 / 總指揮 | 協調、任務分發、結論彙總 |
| Sector Analyst | 行業研究員 | 基本面、行業、估值 |
| Trader | 交易員 | 技術分析、进出场点 |
| Macro Strategist | 宏觀策略師 | 宏觀經濟、政策、跨資產 |
| Intelligence Officer | 情報官 | 新聞、情緒、消息面 |
| Risk Officer | 風控官 | 風險評估、倉位建議 |
| Quant Strategist | 量化策略師 | 量化信號、統計分析 |

## 配置更新

```yaml
agents:
  chief_strategist:
    enabled: true
    token_env: "CHIEF_STRATEGIST_TOKEN"
    is_leader: true                    # ⬅️ 標記為領導者
    team_channel_id: "團隊討論頻道ID"   # ⬅️ 所有 Agent 報告的頻道
    watch_symbols:
      - SPY
      - QQQ

  # 其他 Agent...
```

## 這個架構的優點

1. **透明** - 用戶能看到每個環節的分析過程
2. **專業分工** - 每個 Agent 專注自己的領域
3. **交叉驗證** - 不同視角的分析可以互相印證或矛盾
4. **可擴展** - 新增 Agent 只需要加 Skills 配置
